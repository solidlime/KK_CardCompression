# KK Archive

コイカツ用キャラカード、シーンデータカード、衣装カードの圧縮・解凍ツール。
`Save Load Compression/KK_SaveLoadCompression.dll` と互換性のある実装を目指す。

## 概要
- **対応ファイル**: キャラカード（`.png`）、コーデデータ（`.png`）、Studioシーンデータ（`.png`）
- **主な機能**: フォルダ一括処理、ドラッグ＆ドロップ対応、圧縮・解凍、リッチなGUI
- **特長**: 完全互換、安全設計（自動バックアップ・整合性チェック・ロールバック）、インストール不要

## 使い方
### 単一ファイル処理
1. ファイルをドラッグ＆ドロップ
2. 「自動判定」または「圧縮 / 解凍」を選択
3. 実行 → 出力ディレクトリに同名ファイルを出力

### フォルダ一括処理
1. フォルダをドラッグ＆ドロップ（全対象ファイルをリストアップ）
2. 「自動判定」または「圧縮 / 解凍」を選択
3. 実行 → 階層を維持して出力ディレクトリに出力
