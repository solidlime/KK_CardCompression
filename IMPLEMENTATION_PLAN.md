# 実装プラン: Koikatsu カード圧縮ツールの修正

## 問題点
1. **互換性の問題**: 参照リポジトリは LZMA (101) のみサポート。Zstd (102/103) は現在のプラグイン依存。
2. **PNG チャンク処理**: `SkipPng` の実装が参照リポジトリと微妙に異なる。
3. **トークン判定**: `GuessToken` が `100` 以外をすべて Studio と判定しているが、参照リポジトリは `101` と `100` のみをチェック。
4. **検証処理の欠如**: 参照リポジトリには圧縮後の検証処理があるが、現在のプロジェクトにはない。

## 修正内容

### 1. `CompressionService.cs` の修正
- `GuessToken` の修正: `100` と `101` のみをチェックし、それ以外は Studio と判定（参照リポジトリと一致）。
- `GuessCompressed` の修正: `100` と `101` のみをチェック（参照リポジトリと一致）。
- 圧縮後の検証処理を追加（オプション）。

### 2. `KK_CardCompressionPlugin.cs` の修正
- Zstd 展開処理の改善: エラーハンドリングを強化。
- LZMA 展開処理の追加（オプション）: 参照リポジトリなしでも動作するように。

### 3. GUI アプリの修正
- 圧縮アルゴリズムの選択UIを追加。
- デフォルトを LZMA (101) に変更（互換性確保）。

### 4. テスト
- 既存のカードファイルで圧縮・展開テスト。
- 参照リポジトリとの互換性テスト。

## 実装順序
1. `CompressionService.cs` の修正
2. `KK_CardCompressionPlugin.cs` の修正
3. GUI アプリの修正
4. テスト
